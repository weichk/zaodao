一、总则
1.1 导游用户在注册之前，请仔细阅读本协议，并同意遵守本协议后方可成为注册用户。一旦注册成功，将具有法律效力，用户与早导网之间自动形成协议关系，并受本协议的约束。导游用户在使用早导网站提供的服务或产品时，应当同意接受本协议后方能使用。
1.2 为了适应国家法律法规的规定或者合理调配资源，早导网可能随时更新本协议，用户应当及时关注并同意，早导网不承担通知义务。早导网的通知、公告、声明或其它类似内容是本协议的一部分。
二、服务内容
2.1 早导网将根据不同游客的个性化旅游需求，提供一对一的导游服务；网站同时设有旅游产品，所有服务须在网站上下单交易。
2.2 早导网为导游用户提供免费展示平台，导游在注册后尽量完善个人真实的信息，以提高自己的关注度。游客在浏览网站时，可能会根据其个人喜好选择相对应的导游服务。
2.3 导游用户提供导游服务时，产生的所有费用费用均应由导游用户个人负担，导游用户从金融平台提取现金的，应当按照金融平台的费率支付费用。
2.4 导游用户享有在平台上接受游客预定的权力，也有更新用户主页内容的权力，并享有为游客推荐旅游线路的权力。导游用户在根据游客需求定制旅游服务产品时，需报经网站审核通过后方可使用。严禁导游用户将未经审核的旅游产品提供给游客，由此产生的一切不利后果由导游用户个人承担。早导网有权对导游用户进行注销用户资格和其它相关处罚处理。
三、具体服务流程
注册-----认证----通过审核----接单-----服务确认---服务中----服务结束确认----平台提现
3.1 在早导网注册导游用户并完善相关信息。
3.2 完善个人信息，上传个人照片，导游证和身份证正反面照片，等待后台进行审核。
3.3 审核通过，游客页面可以显示导游用户，即可接单。
3.4 收到早导网发送的游客预定订单信息。
3.5 在工作时间2个小时之内，回电游客，再次确认服务细节。
3.6 根据游客的要求进行导游服务。
3.7 服务结束后，确认服务完成。
3.8 根据金融平台的规则提取现金。
四、用户注册
4.1 导游员在早导网注册账号，须持有国家旅游局颁发的导游资格证书。早导网注册系统将审核确认注册人身份，审核通过后方可成为早导网导游用户，并可获得早导网提供的产品和服务权限；未通过认证的用户不能享受和使用早导网提供的产品和服务。
4.2 实名制注册。用户注册时应按规定使用真实姓名、身份证号及导游证号。
4.3 用户有义务保证密码和账号的安全。因使用账号和密码导致的任何损失或损害，由用户自行承担全部责任，本站不承担任何责任。如用户发现账号遭到未授权人使用或发生其他安全问题，应立即修改账号密码并妥善保管，如有必要，请通知早导网。因黑客行为或用户的保管疏忽导致账号非法使用，早导网不承担任何责任。

  • (4)煽动民族仇恨、民族歧视，破坏民族团结的；
  • (5)破坏国家宗教政策，宣扬邪教和封建迷信的；
  • (6)散布谣言，扰乱社会秩序，破坏社会稳定的；
  • (7)散布淫秽、色情、赌博、暴力、恐怖或者教唆犯罪的；

  • (9)煽动非法集会、结社、游行、示威、聚众扰乱社会秩序的；
  • (10)以非法民间组织名义活动的；

5.3 导游用户在服务过程中要维护早导网和游客的合法权益，如有违反以下情形，早导网有权对导游用户中止协议：
  • (1) 导游用户为游客提供服务时，不得欺骗、胁迫旅游者消费与其它经营者串通欺骗、胁迫旅游者到其消费场所消费。
  • (2) 导游用户进行带团活动时，不得以明示或暗示的方式向游客索取小费。
  • (3) 为保证预定旅行线路的合法化，导游用户不得使用早导网以外的旅游产品，遇特殊情况网站不能及时增补时，需向网站进行申请，产品经认可后方可使用。

  • (5) 不得有损害早导网的任何行为。
5.4 导游用户与游客的联系方式除电话外，应当使用早导网的对话框、QQ、微信等聊天工具，一旦产生纠纷可留取聊天记录作为法律凭证。
5.5 导游用户接受游客预订，预定程序务必在网站上完成，不得私自与游客进行网站外交易，否则早导网可视此行为为非法操作，由此产生的一切后果和损失将由导游用户承担。
